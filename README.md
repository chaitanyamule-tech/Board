KANBAN TASK BOARD (Drag & Drop)

A simple yet powerful Kanban Task Management Board built using HTML, CSS, and Vanilla JavaScript.
This project supports drag-and-drop task movement, persistent storage using localStorage, and real-time task count updates, making it a practical example of real-world frontend state management.
